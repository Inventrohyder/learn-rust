## Instal Rust

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Confirm installation

```bash
rustc --version
```
> rustc 1.65.0 (897e37553 2022-11-02)

```bash
cargo --version
```
> cargo 1.65.0 (4bc8f24d3 2022-10-20)
